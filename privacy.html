Privacy Policy

Data Collection

RequestBlocker collects minimal data necessary to provide our security services. This includes:

  • Request logs (IP addresses, timestamps, request headers)
  • User account information
  • System configuration settings

Data Usage

We use collected data to:

  • Improve threat detection
  • Generate analytics reports
  • Maintain system security
  • Optimize blocking rules

Data Retention

Logs are retained for 30 days by default. Premium users can configure custom retention periods. Account data is maintained until account deletion.

Your Rights

You have the right to:

  • Access your data
  • Request data deletion
  • Export your data
  • Modify retention settings

For privacy-related inquiries, please contact our Data Protection Officer at privacy@requestblocker.com
