Get Started with RequestBlocker

Quick Setup Guide

  1. Install RequestBlocker using npm:
    npm install request-blocker
  2. Configure your API key:
    RB_API_KEY=your_api_key_here
  3. Initialize the monitor in your application:
    require('request-blocker').init();
